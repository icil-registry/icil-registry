# ICIL Registry — Version 1 Specifications

This repository contains the official locked specification documents for
the **Influencer Integrity & Credibility Ledger (ICIL)** v1.0.

ICIL is a neutral, informational public registry for evidence-based
credibility signals related to influencer identities. It does not endorse,
approve, certify, or guarantee any entity.

The following specification documents define ICIL v1:

- `LOCKED_STATE.md` — Locked core rules
- `SCORING_METHODOLOGY_v1.md` — Scoring principles
- `EVIDENCE_SPECIFICATION_v1.md` — Evidence structure
- `DISPUTE_PROCESS_v1.md` — Dispute procedures
- `GOVERNANCE_CHARTER_v1.md` — Governance rules

This repository is intended to provide a permanent, auditable baseline
for ICIL v1. Future versions will require version increments and public
documentation.

**ICIL v1 is locked and immutable.**
